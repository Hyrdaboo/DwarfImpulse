<h4 align="center">An easy to use camera shake plugin for Godot.</h4>
<strong>Note:</strong> this plugin is C# only.

<h3>Features:</h3>

* Easy to use API
* Predefined shake algorithms
* Easily extensible
* Supports both 2D and 3D
